This backend includes:

1. **User Management**:
   - Registration with roles (admin, driver, passenger)
   - Login with JWT authentication
   - Profile management

2. **Vehicle Management**:
   - CRUD operations for vehicles
   - Driver assignment to vehicles
   - Status tracking (active/maintenance/inactive)

3. **Route Management**:
   - Create and manage bus routes
   - Track start/end locations and distances

4. **Trip Management**:
   - Schedule trips with routes and vehicles
   - Track trip status (scheduled/on_route/completed/cancelled)
   - Manage available seats

5. **Booking System**:
   - Passenger booking functionality
   - Seat availability checking
   - Booking status management

6. **Security**:
   - Role-based access control
   - JWT authentication
   - Input validation

7. **File Uploads**:
   - Support for vehicle and driver document uploads
